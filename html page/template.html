Bootcamp 2021: Sand Game

My bootcamp 2021 project, a Powder Game inspired interactive sandbox, based on simulating two types of particles: sand, and water. This project makes significant use of classes and object oriented programming, as well as loops, array lists and light mathematics. The user can press the left or right mouse button to place sand or water particles, respectively. Initially, more particle classes were planned, however the difficulty to implement them grew exponentially, as they all had to have different behaviors when interacting with every other particle class, as well as its own. Sand piles up on itself, and sinks to the bottom of the water, thus pushing the water particles up. Water spills out onto the project area, taking the shape of its container, as it does in real life. Lastly, the game uses a grid system that can be scaled up and down and dynamically adheres to different screen sizes and different grid resolutions.
